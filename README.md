## Dallas Building Permits
Processing [Dallas Bulding Permits](https://developdallas.dallascityhall.com/) has a been an issue for the city Dallas. A massive [backlog](https://www.dallasobserver.com/news/dallas-permit-process-delays-11968181) of pending buidling permits has slowed down the city's ability to add desparately needed limits. The focus of this project is to harvest city permits data and create a Flask API with city building data.
